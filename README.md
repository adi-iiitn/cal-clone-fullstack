\# 📅 Cal.com Clone - Full Stack Calendar Booking App



A modern calendar booking application built with Next.js, Express, PostgreSQL, and Prisma.



\## 🚀 Features



\- ✅ Real-time availability checking

\- ✅ Conflict prevention (no double bookings)

\- ✅ Email notifications (confirmation \& cancellation)

\- ✅ Timezone handling

\- ✅ Modern UI with Tailwind CSS

\- ✅ Type-safe API with TypeScript
