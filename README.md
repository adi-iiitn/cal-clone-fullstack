

# 📅 Cal Clone - Full-Stack Meeting Scheduler

A modern, full-featured meeting scheduling application inspired by Cal.com, built with Next.js, Express, PostgreSQL, and Prisma.

## ✨ **Features**

### 🎯 **Core Functionality**
- ✅ **Event Type Management** - Create and manage different meeting types (30min, 60min, custom)
- ✅ **Real-time Availability Checking** - Smart time slot generation based on existing bookings
- ✅ **Conflict Prevention** - No double bookings with intelligent scheduling
- ✅ **Email Notifications** - Beautiful HTML email confirmations and cancellation notices
- ✅ **Timezone Support** - Automatic timezone detection and conversion
- ✅ **Responsive Design** - Mobile-first UI that works on all devices
- ✅ **Modern UI** - Built with Tailwind CSS and Shadcn/ui components

### 🔧 **Technical Features**
- ✅ **Type-Safe** - Full TypeScript implementation (frontend + backend)
- ✅ **RESTful API** - Clean, documented API endpoints
- ✅ **Database Migrations** - Prisma ORM with PostgreSQL
- ✅ **CORS Configured** - Secure cross-origin resource sharing
- ✅ **Environment Variables** - Secure configuration management
- ✅ **Error Handling** - Comprehensive error handling and logging
